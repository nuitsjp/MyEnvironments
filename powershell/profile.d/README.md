# profile.d - 即時ロード設定

このディレクトリ内の `.ps1` ファイルは、PowerShell 起動時に自動的に読み込まれます。

## ファイル構成

| ファイル | 役割 | 管理 |
|---------|------|------|
| `00-env.ps1` | 環境変数・共通フラグ | Git |
| `10-alias.ps1` | alias 定義 | Git |
| `20-functions.ps1` | 軽量関数 | Git |
| `30-tools.ps1` | CLI ツール補助（copilot / codex / git 等） | Git |
| `90-local.ps1` | ローカル専用設定 | **Git 管理外** |

## ルール

- **軽量・短く**: 起動時に毎回実行されるため、重い処理は避ける
- **副作用明確**: 環境変数やグローバル設定の変更は慎重に
- **命名規則**: 数字プレフィックスで読み込み順序を制御

## ローカル設定

個人環境固有の設定は `90-local.ps1` に記述してください。  
このファイルは `.gitignore` により Git 管理外となります。
