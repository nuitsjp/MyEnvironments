# Skill Best Practices Applier

スキルをベストプラクティスに照らして分析し、改善を適用します。

## ワークフロー

### Step 1: 改善対象のスキルを選択

引数でスキルパスが指定されていればそれを使用。
指定がなければ、`.claude/skills/` 内のスキル一覧から選択してもらう。

### Step 2: 現在の構造を分析

スキルのファイルを読み込み、以下を確認:
- 現在のSKILL.mdの行数
- ワークフローが分離されているか
- 参照構文（`@` vs マークダウンリンク）

### Step 3: チェックリストと照合

以下の重要項目を評価:

**必須項目**:
- SKILL.mdが30行未満
- マークダウンリンク（`@`構文ではない）
- ワークフローが別ファイル
- バイリンガルな説明

### Step 4: 結果を報告

ユーザーに提示:
- 現在の行数 vs 目標（30行未満）
- パスした項目
- 改善が必要な項目（具体的な修正案付き）

### Step 5: 改善を適用

ユーザーの確認後:
1. 必要に応じてworkflow.mdを作成
2. SKILL.mdを最小限の構造に簡素化
3. `@`参照をマークダウンリンクに変換
4. 説明をバイリンガルキーワードで更新
