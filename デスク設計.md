## デバイス一覧

| デバイス | 接続方式 | 用途 |
|---------|--------|------|
| デスクトップPC1 | DisplayPort + HDMI | メイン機器（KVM経由） |
| デスクトップPC2 | HDMI + 無線USB | サブ機器 |
| ノートPC | Type-C | モバイル機器（KVM経由） |
| ミニPC | Type-C（KVM経由） | 小型機器 |
| KVM | - | 入出力切り替え（映像・Type-C） |
| HDMI to Type-C変換 | HDMI to Type-C（PC1・KVM・ポータブルモニター） | 変換・ハブアダプタ |
| HDMIスイッチ | HDMI入出力 | サブディスプレイ切り替え |
| Bluetoothドングル1 | USB（USBハブ経由） | Bluetooth接続用 |
| Bluetoothドングル2 | USB（USBハブ経由） | Bluetooth接続用 |
| 無線ドングル | 無線＋USB（USBハブ経由） | ヘッドセット用無線ドングル |
| ヘッドセット | 無線 | USBハブ経由のオーディオデバイス |
| USBハブ | USB（KVM経由） | 入力機器用ハブ |
| メインディスプレイ | DisplayPort（KVM経由） | 4K, 43インチ |
| サブディスプレイ | HDMI（HDMIスイッチ経由） | 共有画面 |
| ポータブルモニター | Type-C（HUB1経由・映像・タッチ・電源） | サブ画面 |
| キーボード | USB（USBハブ経由） | KVM経由の入力デバイス |
| マウス | 無線（Logi Bolt経由） | KVM経由の入力デバイス |
| Logi Bolt（セット1用） | 無線＋USB（USBハブ経由） | セット1用ワイヤレスレシーバー |
| Webカメラ | USB（USBハブ経由） | KVM経由の入力デバイス |
| キーボード2（無線） | 無線（Logi Bolt経由） | セット2用入力デバイス |
| マウス2（無線） | 無線（Logi Bolt経由） | セット2用入力デバイス |
| Logi Bolt | 無線＋USB | セット2用ワイヤレスレシーバー |
